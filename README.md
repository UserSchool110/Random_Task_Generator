# Random Task Generator — Генератор случайных задач

## Информация об авторе

## Описание программы

**Random Task Generator** — это GUI-приложение для генерации случайных задач и отслеживания истории выполненных заданий. Приложение позволяет генерировать случайные задачи из списка, фильтровать их по категориям, сохранять и загружать данные в формате JSON, а также управлять списком задач.

### Возможности:  
- ✅ Генерация случайной задачи из предопределённого списка
- ✅ Добавление собственных задач с указанием категории
- ✅ Фильтрация истории по категориям (учёба, спорт, работа)
- ✅ Сохранение истории в JSON-файл
- ✅ Автоматическая загрузка истории при запуске
- ✅ Очистка всей истории с подтверждением
- ✅ Валидация вводимых данных (проверка пустых полей)

## Инструкция по запуску

### Требования:  
- Python 3.6 или выше  
- Установленная библиотека Tkinter (обычно идёт в комплекте с Python)

### Запуск:  
1. Откройте терминал или командную строку  
2. Перейдите в директорию проекта:  
   ```bash
   cd random_task_generator
   ```  
3. Запустите приложение:  
   ```bash
   python random_task_generator.py
   ```

## Примеры использования

### Генерация задачи:  
1. Нажмите кнопку **«сгенерировать задачу»**  
2. В верхней части окна появится случайная задача (например, «прочитать статью [учёба]»)  
3. Задача автоматически добавится в историю внизу окна

### Добавление новой задачи:  
1. В поле ввода введите название задачи (например, «Выучить новый язык»)  
2. Выберите категорию через фильтр (учёба, спорт, работа)  
3. Нажмите кнопку **«добавить»**  
4. Задача появится в общем списке и будет доступна для генерации

### Фильтрация:  
- **По категории:** Выберите нужную категорию («все», «учёба», «спорт», «работа»)  
- История автоматически обновится, показывая только задачи выбранной категории

### Сохранение и загрузка:  
- Данные **автоматически сохраняются** в файл `task_history.json` при каждом добавлении задачи в историю  
- При следующем запуске история автоматически загружается из файла  
- Список задач пользователя сохраняется в файл `tasks.json`

### Удаление:  
- Нажмите кнопку **«очистить историю»** и подтвердите действие — вся история будет удалена  
- Обратите внимание: удаление истории не удаляет сами задачи из списка доступных

## Тестовые данные (примеры для проверки)

Предопределённые задачи, доступные при первом запуске:

| Задача | Категория |
|--------|-----------|
| прочитать статью | учёба |
| сделать зарядку | спорт |
| написать отчёт | работа |

После добавления попробуйте:  
- Сгенерировать несколько задач и проследить за появлением истории  
- Отфильтровать историю по категории «учёба»  
- Добавить собственную задачу и сгенерировать её  
- Закрыть приложение, затем открыть снова — история должна сохраниться

## Валидация данных

Приложение проверяет:  
- Поле добавления задачи не может быть пустым  
- Нельзя добавить дубликат существующей задачи  
- Перед очисткой всей истории требуется подтверждение  
- При отсутствии задач в списке генерация невозможна (выводится предупреждение)
